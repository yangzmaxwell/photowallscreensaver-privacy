照片墙风格屏幕保护程序 v1.0.4

-简介-
照片墙风格屏幕保护程序是一款实现图片动态更换加载效果的屏幕保护程序。本屏保尤其适合电影海报的自动替换风格屏保。

-核心功能-
1.屏幕保护启动实现4种加载效果
2.屏保运行期间，图片切换支持6种动态过渡效果
交叉隐现
翻转
上下左右依次替补

-使用方法-
安装与设置
将zip包解压至任意位置，右键点击安装即可运行。
首次运行需要制定图片位置，如果是电影海报图片，推荐2行5列，或3行8列。
也可将PhotoWallScreenSaver.scr复制到C:\Windows\System32\
在“设置”中“更改屏幕保护设置”中选择PhotoWallScreenSaver

如果Windows重启后屏保设置丢失，请运行gpedit.msc，进入“本地组策略”，选择用户配置>管理模板>控制面板>个性化，点击“强制使用特定的屏幕保护程序”，选择“已启用”，可执行的屏保名称输入“%Systemroot%\System32\PhotoWallScreenSaver.scr”，或者本屏保所在位置。

支持图片格式包括：.jpg、.jpeg、.png、.bmp、.gif、.webp。

-注意事项-
请确保指定的图片文件夹中包含有效图片文件。
动态效果的流畅度可能因图片分辨率、计算机性能略有差异。
因采用单一文件打包方式，如遇到杀毒软件、防护软件提醒，请添加信任。

Photo Wall Screensaver v1.0.4

- Introduction -

The Photo Wall Screensaver is a screensaver that implements dynamic image loading effects. It is particularly suitable for an automatic replacement style featuring movie posters.

- Core Features -

Startup Effects: Features 4 loading effects upon screensaver activation.
Transition Effects: Supports 6 dynamic transition effects for image switching during operation, including:
Cross Fade
Flip
Sequential Slide (Top/Bottom/Left/Right)

- How to Use -

Installation & Setup

Extract the zip file to any location and right-click to install/run.
On the first run, you need to specify the image directory. For movie posters, a layout of 2 rows x 5 columns or 3 rows x 8 columns is recommended.
Alternatively, copy PhotoWallScreenSaver.scr to C:\Windows\System32\.
In Windows "Settings" > "Change screen saver", select PhotoWallScreenSaver.

Troubleshooting (Settings Lost after Restart)

If the screensaver settings are lost after a Windows restart, please run gpedit.msc to open the Local Group Policy Editor. Navigate to User Configuration > Administrative Templates > Control Panel > Personalization. Enable the policy "Force specific screen saver" and enter the path to the executable (e.g., %Systemroot%\System32\PhotoWallScreenSaver.scr or the custom location where you saved the file).

Supported Formats: .jpg, .jpeg, .png, .bmp, .gif, .webp.

- Notes -

Please ensure the specified image folder contains valid image files.
The smoothness of dynamic effects may vary slightly depending on image resolution and computer performance.
Since the software is packaged as a single file, if you receive alerts from antivirus or security software, please add it to the trust/whitelist.
